How It Works
  • When you switch models, the extension extracts keywords from the model name
  • It searches for presets with matching names (e.g., "claude-sonnet-4" → "Claude" preset)
  • If a match is found above the sensitivity threshold, that preset is automatically applied
  • If no match is found, the fallback preset is used instead
  • Example: Any model with "claude" in the name will match a preset named "Claude"
